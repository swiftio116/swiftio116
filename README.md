## 👨‍💻 About me

I am a **Junior iOS Developer** focused on Swift, UIKit and clean mobile architecture.

I build iOS apps with **programmatic UIKit**, **native Auto Layout**, `UITableView`, `UICollectionView`, `URLSession`, `Codable`, `CoreLocation` and **Firebase Auth / Firestore**.

I understand the basics of **OOP**, **SOLID**, **MVC**, **MVVM**, **MVP**, **Coordinator**, protocol-based Dependency Injection and unit testing.

After code review, I improved my projects by removing Storyboards/XIB where needed, moving dependencies into coordinators/containers, adding mock services and writing tests for ViewModel/Presenter logic.

My goal is to join a strong engineering team as an **iOS Intern / Junior iOS Developer**, work on real product tasks, pass code review and grow into an independent iOS developer.
